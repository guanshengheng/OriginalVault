### [小红的奇数奶油球](https://ac.nowcoder.com/acm/contest/127637/D)
搜索是为了预处理得到节点下的子树最大连通分量大小，然后根据这个信息进行条件判断。
大致就有存图、搜索、判断三个阶段。
搜索时核心递归代码如下。
```
void dfs(int u, int p)
{
    fa[u] = p;
    sz[u] = 1;
    for (int v : tr[u])
    {
        if (v == p)
            continue;
        else
        {
            dfs(v, u);
            sz[u] += sz[v];
        }
    }
}
```
